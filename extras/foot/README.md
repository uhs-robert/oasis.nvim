# Foot Terminal Setup

1. Copy the generated `.ini` theme file (e.g., `oasis_lagoon.ini`) to your Foot configuration directory, typically `~/.config/foot/themes/`.
2. Reference the theme in your `foot.ini` configuration file.

### Example

```ini
include=~/.config/foot/themes/oasis_lagoon.ini
```

Or, you can directly use the generated `.ini` file as your main `foot.ini` if you prefer to have a dedicated configuration per theme.
