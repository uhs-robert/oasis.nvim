## Installation

1. Click any theme link above
2. Firefox Color will open with the theme preview
3. Click the **Install** button to apply the theme
4. The theme will be saved to your Firefox profile

## Customization

After installing a theme, you can further customize it using Firefox Color:

1. Open [Firefox Color](https://color.firefox.com)
2. Select your installed Oasis theme
3. Adjust individual colors as desired
4. Save your customized theme

## Textfox - TUI Firefox

Wouldn't it be nice if Firefox looked like a TUI application? Check out [**Textfox**](https://github.com/adriankarlen/textfox) - a TUI (Text User Interface) configuration for Firefox. It's what I use personally and this Oasis design is tailored for it specifically.

Textfox provides a minimal Firefox interface that complements the Oasis color palette perfectly for users who prefer terminal-based workflows.
