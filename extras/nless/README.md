# nless Setup

[nless (nothing-less)](https://github.com/mpryor/nothing-less) is a minimal, keyboard-driven spreadsheet viewer (csv, json, logs, etc) for the terminal.

1. Copy your desired theme file(s) into `~/.config/nless/themes/`

1. In your nless config, set the theme by name (without `.json`):

```
theme = oasis_lagoon_dark
```
