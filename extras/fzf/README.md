# FZF Setup

Add the Oasis theme to your shell configuration by sourcing the theme file.

## Installation

### Bash/Zsh

Add to your `~/.bashrc` or `~/.zshrc`:

```bash
# Source the Oasis FZF theme
source /path/to/oasis.nvim/extras/fzf/oasis_lagoon.sh
```

### Fish

Add to your `~/.config/fish/config.fish`:

```fish
# Source the Oasis FZF theme
source /path/to/oasis.nvim/extras/fzf/oasis_lagoon.sh
```
