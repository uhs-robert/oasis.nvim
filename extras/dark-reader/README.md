# Dark Reader Setup

Dark Reader is a browser extension that applies dark themes to websites. These Oasis themes provide custom color schemes for a consistent experience across the web.

<img src="/assets/screenshots/extras/darkreader.png" width="2534" height="1518" alt="Darkreader"  />

## Installation

1. Install the Dark Reader extension:
   - **Chrome/Edge**: [Chrome Web Store](https://chrome.google.com/webstore/detail/dark-reader/eimadpbcbfnmbkopoojfekhnkhdbieeh)
   - **Firefox**: [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/darkreader/)

2. Choose your preferred Oasis theme file (e.g., `oasis_lagoon.txt`)

## Usage

### Enable the new Dark Reader design prototype (skip if already enabled)

1. Open the Dark Reader extension
2. Click **"More"** then **"All Settings"**
3. Click **"Advanced"** then **"Dev Tools"**
4. Click **"Advanced"** again, then click **"Enable design prototype"**
5. Close and reopen the Dark Reader extension

### Update your Dark Reader settings

1. Open the Dark Reader extension
2. Click **"See all options"** and then click **"Colors"**
3. Copy the color values from your chosen theme file and paste them into the respective fields:
   - **Background**: Core background color
   - **Text**: Main text color
   - **Scrollbar**: Accent color for scrollbars
   - **Selection**: Text selection highlight color
