# Warp Setup

1. Create `~/.warp/themes`
2. Download your desired theme file(s) into the `themes` directory
3. Select your theme in [Warp's settings](https://docs.warp.dev/terminal/appearance/themes_): `Settings > Appearance > Custom Theme`

### Theme Structure

- **Dark themes**: `themes/dark/oasis_<variant>_dark.yaml`
- **Light themes**: `themes/light/<intensity>/oasis_<variant>_light_<intensity>.yaml`
  - Intensity levels: `1` (lightest) to `5` (darkest)

### Example

Download `oasis_lagoon_dark.yaml` to `~/.warp/themes/` and select it from Warp's theme picker.
