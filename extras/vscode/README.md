# Oasis VS Code Theme

Desert-themed colorscheme for VS Code. 96 themes: 16 palette variants × 1 dark + 5 light intensity levels.

## Quick Install (Sideload)

```bash
ln -s /path/to/oasis.nvim/extras/vscode ~/.vscode/extensions/uhs-robert.oasis-nvim-4.0.0
```

Restart VS Code, then `Ctrl+K Ctrl+T` and search **Oasis** to pick a theme from `Themes > Color Theme`.
